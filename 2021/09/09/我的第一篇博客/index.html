HEXO博客搭建流程

发表于2021-09-09|更新于2021-09-25|技术
|阅读量:
Hexo博客是现在比较流行的表态博客网站，没有后台，文章的管理需要先在本地处理，之后用命令行推送到Github或Gitee上面进行更新后才能被访问。以下来博客的搭建流程：
  1. 安装nodjs环境，这个可以到官方网站去下载
  2. 安装git环境,去官方网站下载
  3. 去Hexo官方网站安装Hexo
  4. 所有环境都安装完毕后，在终端或CMD命令行中查看环境是否安装成功
    • 查看nodejs是否安装成功：node -v
    • 查看git是否安装成功：git -version
    • 查看Hexo是否安装成功：hexo -v
    • 如果上面的都正常显示版本号，说明安装成功
  5. 安装淘宝镜像
    1. 命令行中运行：npm install -g cnpm --registry=https://registry.npm.taobao.org
  6. 部署Hexo环境
    1. 安装Hexo 依赖：npm install hexo-cli -g
    2. 在博客的文件夹内运行：hexo init
      • 也可以在外层文件夹运行：hexo init blog
    3. 在博客文件夹内运行：npm install
    4. 安装hexo-deployer-git插件：npm install --save hexo-deployer-git
    5. 部署完成，接下来可以通过命令操作
      • hexo cl ——清理缓存
      • hexo g ——生成静态网页
      • hexo s ——部署本地服务器
      • hero d ——推送到Github或Gitee
      • hexo n “名称” ——创建博客，在source/_post/ 文件夹内就会生成名称.md的文件，可以直接用markdown写博客
